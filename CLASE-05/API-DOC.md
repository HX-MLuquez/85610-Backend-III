# Documentación de la API RESTful - AdoptMe API

## Descripción General
AdoptMe API es una API RESTful diseñada para gestionar la adopción de mascotas. Permite a los usuarios ver, agregar, actualizar y eliminar información sobre mascotas disponibles para adopción.

## Endpoints
### Obtener todas las mascotas
- **URL:** `/api/pets`
- **Método:** GET
- **Descripción:** Recupera una lista de todas las mascotas disponibles para adopción.

### Obtener una mascota por ID
- **URL:** `/api/pets/{id}`
- **Método:** GET
- **Descripción:** Recupera la información de una mascota específica utilizando su ID.
